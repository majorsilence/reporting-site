客户端 SDK

在任意语言中使用 Majorsilence Reporting

每个 SDK 都封装了同一个 RDL 引擎——通过子进程、AOT 二进制文件或原生 FFI——让你无需在调用处依赖 .NET,即可用现有技术栈生成报表。

Python majorsilence-reporting
Python SDK

Python 3.10+。除标准库外无其他运行时依赖。支持子进程、AOT 二进制文件,或进程内 ctypes FFI。

子进程 AOT 二进制 ctypes FFI

PHP majorsilence/reporting
PHP SDK

PHP 8.1+。无 Composer 运行时依赖。支持子进程、AOT 二进制文件,或进程内 PHP FFI。可直接将报表流式输出到浏览器。

proc_open AOT 二进制 PHP FFI

Ruby majorsilence_reporting
Ruby SDK

Ruby 3.0+。无 gem 运行时依赖。支持子进程、AOT 二进制文件,或进程内 Fiddle FFI。可与 Rails、Sinatra 和 Rack 配合使用。

子进程 AOT 二进制 Fiddle FFI

Rust majorsilence-reporting
Rust SDK

Rust 2021 版。仅有一个依赖项(libloading)。仅使用原生 FFI——无需子进程,无需 .NET 运行时。线程安全、基于 Arc 的库句柄。

libloading FFI Arc + Send + Sync

全部格式,全部模式

每个 SDK 在所有集成模式下都支持 pdf、xlsx、csv、xml、rtf、tif 和 html。AOT 二进制模式和原生 FFI 模式还可解锁 xlsx_table、tifb 和 mht。所有原生 FFI 模式都可以通过 add_data() 使用内存数据集(无需数据库)。
