SDKs de cliente

Usa Majorsilence Reporting desde cualquier lenguaje

Cada SDK envuelve el mismo motor RDL — subproceso, binario AOT o FFI nativa — para que puedas generar informes desde tu stack actual sin depender de .NET en el punto de llamada.

Python majorsilence-reporting
SDK de Python

Python 3.10+. Sin dependencias en tiempo de ejecución más allá de la biblioteca estándar. Subproceso, binario AOT, o FFI ctypes en proceso.

Rust majorsilence-reporting
SDK de Rust

Edición 2021 de Rust. Una sola dependencia (libloading). Solo FFI nativa — sin subprocesos, sin runtime de .NET. Handle de biblioteca seguro para hilos, respaldado por Arc.

Todos los formatos, todos los modos

Todos los SDK admiten pdf, xlsx, csv, xml, rtf, tif y html en todos los modos de integración. El binario AOT y los modos FFI nativos habilitan además xlsx_table, tifb y mht. Los conjuntos de datos en memoria (sin necesidad de base de datos) están disponibles en todos los modos FFI nativos mediante add_data().
